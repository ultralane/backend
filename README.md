# Relayer

### Installation

```
git clone https://github.com/ultralane/relayer
cd relayer
npm i
```

### Environment variables
Create a .env file and add the following data
```
APP_NAME=Ultralane
PORT=8000
MONGO_URI=
PRIVATE_KEY=
ALCHEMY_API_KEY=
```

> Note: PRIVATE_KEY mentioned here is the owner of the ultralane contracts

### Running it locally

```
npm run dev
```
